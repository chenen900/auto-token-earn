## 九、新电脑启动步骤

1. 复制 `交接文档-出差期间.md` + `.env` + `.claude/` 文件夹 到新电脑
2. `git clone https://github.com/chenen900/auto-token-earn.git`
3. 把 `.env` 放进去
4. **安装184个AI Agent**（开源Agent库，来自GitHub）：
```bash
git clone https://github.com/msitarzewski/agency-agents.git /tmp/agency-agents
cd /tmp/agency-agents && bash scripts/install.sh --tool claude-code
```
这会把Agent安装到 `~/.claude/agents/`，之后Claude Code就能调用全部184个Agent。
5. **安装Agent技能包**（GSAP+Anthropic技能模板）：
```bash
mkdir -p .claude/skills
cp -r gsap-skills/skills/* .claude/skills/
cp -r anthropic-skills/skills/* .claude/skills/
```
5. 对Claude Code说：`读取交接文档-出差期间.md，了解项目状态`

## 九、经验教训

### 技术坑
1. **curl传中文会乱码** → 测试API用Node.js发请求，不要用curl
2. **JS和HTML必须分开** → 内联脚本多次编辑会损坏，独立.js文件最稳
3. **阿里云FC强制下载HTML** → 不适合网页，只做API。网页放Render
4. **别依赖被墙的CDN** → GSAP(Cloudflare)国内加载失败，用纯CSS/JS替代
5. **Render自动部署，阿里云手动上传** → 改网页刷Render，改API才传阿里云
6. **GitHub SSH不稳定时换HTTPS** → 多一个推送方式备用

### 平台坑
7. **dealwork.ai供给>需求** → 投标多中标少，耐心等平台成长
8. **AgentHansa新号5天限制** → 前5天只能Response型，第6天Personal Task解锁
9. **x402 v2格式严格** → amount整数、resource对象、需要Base钱包
10. **Agentic Market验证器缓存** → 改完等几分钟再验证

### 策略教训
11. **多条小线并行 > 一条深线** → 10条×$0.50=$5/天
12. **免费工具箱引流 > 按次收费** → 用户反感扣费，会员+广告更可持续
13. **公共物品养生态，基建API赚钱** → 规则库免费→合规API收费
14. **中国广告法合规是蓝海** → 全网没有第二家做这个
