## 投资逻辑

```
公共物品（免费）
    ↓
降低 AI 产业门槛 → 更多人做 AI 工具
    ↓
他们需要合规能力 → 调我们 API
    ↓
基础设施层（付费 API）
    ↓
现金流
```

**花钱养公共物品 → 更多人进 AI 赛道 → 更多人需要合规 → 我们的 API 调用量上去。**

泡沫不泡沫不重要——**如果泡沫破了，有基础设施的人活下来；如果泡沫变成产业，早期基建者赢最大。**
