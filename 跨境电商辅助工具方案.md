## 可快速开发的工具

### 1. Amazon Listing 合规助手
```
输入：产品标题 + 五点描述 + 关键词
输出：
  ✅ 合规评分
  ⚠️ 违规项（夸大宣传/医疗声称/禁止用语/竞品对比）
  📝 修改建议
  🌍 多语言翻译版
```
**接入方式：** Chrome扩展（在Amazon Seller Central页面直接检查）

### 2. 多平台 Listing 一键生成
```
输入：中文产品名 + 卖点 + 规格
输出：
  → Amazon英文Listing（SEO优化 + FTC合规）
  → Temu英文Listing
  → Shopee 中文/英文Listing
  → 拼多多中文Listing
```
**每次调用：** 翻译1次 + 合规1次 + SEO 1次 = $0.04/平台

### 3. 跨境电商合规规则库
```
平台 × 国家 × 产品类目 = 专属合规规则

Amazon.com  → FTC + FDA + CPSC + Amazon政策
Amazon.co.jp → 日本景表法 + 薬機法 + Amazon政策
Temu         → FTC + 各州消费者保护法
Shopee       → 各国电商法 + Shopee政策
```

### 4. SKILL.md — AI Agent 跨境电商技能包
```
我们发布一个 Claude Code 技能：
  ecommerce-compliance.skill.md

安装了它的 Claude Code 用户说：
  "帮我把这个Listing翻译成英文并检查合规"

Claude Code 自动调：
  翻译API → 合规API → SEO API → 输出最终结果
```

## 从 GSAP Skills 学到的

GSAP（动画库）为 AI Agent 发布了 8 个技能文件，教 AI 怎么写动画代码。这说明：**专业技能 + SKILL.md = AI Agent 能力扩展。**

我们可以做同样的事：

| 我们的技能包 | 内容 |
|------|------|
| `content-compliance-cn` | 中国广告法 + 17平台规则审查 |
| `cross-border-listing` | 跨境电商Listing生成+合规+SEO |
| `api-directory` | API黄页查询 |
| `x402-payments` | AI Agent 付费API接入 |
