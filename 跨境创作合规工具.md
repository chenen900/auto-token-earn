# 跨境创作合规工具 —— 中国创作者出海"守门员"

## 为什么这是个金矿

### 现实痛点

| 问题 | 影响 |
|------|------|
| 中国创作者不理解 YouTube 版权规则 | 音乐侵权 → 版权警示 → 三次封号 |
| 不知道 Fair Use（合理使用）边界 | 引用影视片段 → 侵权下架 |
| 抖音思维直接用 TikTok | 抖音允许的 ≠ TikTok 允许 |
| 不熟悉 FTC 广告披露要求 | 软广不标注 → 罚款 |
| 不懂各平台社区规范 | 内容限流、推荐降权、封号 |
| 出海品牌不了解当地广告法 | 虚假宣传 → 被起诉 |

### 市场信号

- 2026年中国跨境电商交易规模超 **¥20万亿**
- YouTube 中文创作者 **年增 200%**
- TikTok 中国出海品牌 **超 50 万家**
- **没有一个工具**专门帮中国创作者做海外平台合规

## 产品形态

### 核心功能：AI 合规审查 API

```
输入：脚本/文案 + 目标平台
输出：合规报告（违规点 + 具体法条 + 修改建议）

覆盖平台：YouTube / TikTok / Instagram / Facebook / Amazon
覆盖法规：FTC广告法 / DMCA版权法 / 平台社区规范 / GDPR
```
