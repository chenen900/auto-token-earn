## 集成方式（按用户画像）

### 画像 A：跨境电商新手
> "我想在亚马逊上卖东西"

| 触点 | 怎么用 |
|------|--------|
| Chrome 扩展 | 在亚马逊后台写 Listing 时，点一下按钮直接查 |
| 网页版工具 | 粘贴文案 → 一键查 → 复制结果 |
| API | 如果自己做 ERP 系统，直接调 |

**获利点：** 每个 Listing 3 次调用（翻译+SEO+合规）= $0.04。一个人 50 个产品 = $2。一次付费就够了。

---

### 画像 B：小红书创作者
> "帮我做小红书自动发文工具"

| 触点 | 怎么用 |
|------|--------|
| 网页版工具 | 写完种草笔记 → 粘贴检查 → 发布 |
| npm CLI | `npx mediacraft-check "文案" --platform xiaohongshu` |
| VS Code 扩展 | 写 Markdown 时右边栏实时显示问题 |
| MCP Server | Claude Code 里说"审查这段小红书文案" |

**获利点：** 创作者日更 1-2 篇，每次发前查一次 = $0.02/天 = $0.60/月。便宜到不用想。

---

### 画像 C：中国创作者出海
> "我把视频搬到 YouTube，怕违规"

| 触点 | 怎么用 |
|------|--------|
| Chrome 扩展 | 在 YouTube Studio 写标题描述时实时检查 |
| API | 脚本批量翻译+检查 |
| 网页版工具 | 一站式：翻译 → 合规 → SEO → 发布 |

**获利点：** 一条视频 4 次调用（翻译+合规+SEO+报告）= $0.09。周更 3 条 = $1.08/月。比请一个懂海外合规的运营便宜 1000 倍。

---

### 画像 D：AI 工具开发者
> "我在做一个内容生成工具，需要合规能力"

| 触点 | 怎么用 |
|------|--------|
| x402 API | AI Agent 自动发现、自动付费、自动调用 |
| npm SDK | `npm install mediacraft-sdk` → 一行代码 |
| REST API | 标准 HTTP，Swagger 文档自动生成 |

**获利点：** 一个开发者工具月活 1000 用户 = 10000 次调用/月 = $100-200/月。10 个这样的开发者 = $1000-2000/月。

---

### 画像 E：Claude Code / VS Code 用户
> "我在编辑器里写东西，想直接查"

| 触点 | 怎么用 |
|------|--------|
| MCP Server | Claude Code 直接调，零配置 |
| VS Code 扩展 | 选中文字 → Ctrl+Shift+C → 结果在下方面板 |
| npm CLI | `npx mediacraft-check` → 终端输出 |

**获利点：** 免费额度 100 次用完后，每次 $0.01-0.02。VS Code 市场 2000 万开发者，假设 0.1% 转化 = 2 万用户 × $0.50/月 = $10,000/月。

## 为什么走调用量而不是订阅制

| | 订阅制 | 按量付费 |
|------|--------|---------|
| 用户心理 | "每月都要付，用不用都得交" | "用一次才付一次，不用不花钱" |
| 决策门槛 | 高（需要审批、预算） | 零（$0.01 不需要想） |
| 适合我们 | ❌ | ✅ |
| 增长方式 | 销售驱动 | 产品驱动，靠分发 |
| AI Agent 能用吗 | ❌（要人类签约） | ✅（自动付费） |

**我们的定位：AI 内容经济的"电费"——按用量收，便宜到没人想省。**
