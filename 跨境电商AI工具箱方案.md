## 我们能建的 6 个工具

### 1. Listing 生成器（已有能力直接包装）

```
输入：1688商品链接
输出：
  → 自动抓取标题/图片/规格
  → AI 翻译成英文（$0.01）
  → SEO 优化标题和关键词（$0.01）
  → 合规审查（$0.02）
  → 适配 Amazon / Temu / Shopify 格式

一个 Listing 总成本：$0.04
```

### 2. 合规雷达

```
输入：目标市场（美国/欧洲/日本）+ 产品类目
输出：
  → 这个市场需要什么认证（FDA/FCC/CE/CPSC...）
  → 广告文案能不能这样说
  → 产品图片有什么要求
  → 标签/包装规范
```

### 3. 利润计算器

```
输入：进货价 + 重量 + 目标市场
计算：
  → 物流成本（云途/4PX 报价）
  → 平台佣金（Amazon 15%/Temu 免费）
  → 广告预算建议
  → 到岸总成本
  → 建议售价 + 利润率
```

### 4. 1688 商品翻译导入器

```
输入：1688 商品链接
自动：
  → 抓取商品信息
  → 翻译标题+描述
  → 检查是否包含禁止出口品类
  → 生成英文/日文/西语 Listing
  → 一键导出 CSV
```

### 5. 广告文案生成 + 合规

```
输入：产品信息 + 目标平台
输出：
  → 5 条 Facebook 广告文案（英语）
  → 5 条 Google Shopping 标题
  → 3 条 TikTok 口播脚本
  → 每条都检查 FTC/EU 广告法规
```

### 6. 入门知识库（免费公共物品）

```
内容：
  → "我该怎么开始" 3 步指南
  → 各平台对比（Amazon vs Temu vs 独立站）
  → 必办手续清单（营业执照、IOSS、商标）
  → 常见错误和罚款案例
  → 全部中英双语
```

## 技术方案

```
前端：一个网页（HTML + JS）
    ├── Listing 生成器
    ├── 合规雷达
    ├── 利润计算器
    ├── 1688 导入器
    ├── 广告文案生成器
    └── 知识库

后端：调我们的现有 API
    ├── translate API
    ├── compliance-check API
    ├── seo-optimize API
    └── API directory（查竞品用）
```

全部静态部署到 Render（免费），后端调已有 API。
