## 五、新电脑启动步骤

### 1. 复制文件
```
从旧电脑复制到新电脑：
- d:\ai赚钱体系\.env
- d:\ai赚钱体系\auto-token-earn\data\platform_credentials.json
```

### 2. 克隆仓库
```bash
git clone https://github.com/chenen900/auto-token-earn.git
cd auto-token-earn
# 把 .env 和 platform_credentials.json 放进去
```

### 3. 安装工具
```bash
# 参考 工具安装记录.md 中的完整命令列表
# 核心：
npm install
npm install -g @lucid-agents/taskmarket
taskmarket init
```

### 4. 启动守护
```bash
# 打开 keepalive.html（浏览器）
# 打开 command_monitor.html（浏览器）
# 启动 Chat Bridge
node cmd_chat_bridge.js
```

### 5. 配置 GitHub Secrets
打开 https://github.com/chenen900/auto-token-earn/settings/secrets/actions
确保以下 Secrets 存在：
- `AGENTHANSA_API_KEY`
- `DEALWORK_API_KEY`
- `DEVTO_API_KEY`
- `SUPERTEAM_API_KEY`
- `TOKU_API_KEY`
- `HASHNODE_API_KEY`

### 6. 对 Claude Code 说
```
读取 d:/ai赚钱体系/auto-token-earn/交接文档-v3.md，了解项目状态
```
