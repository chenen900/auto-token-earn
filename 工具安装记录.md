## 安装命令（按顺序复制粘贴）

```bash
# 1. agency-agents (184个Agent)
git clone https://github.com/msitarzewski/agency-agents.git /tmp/agency-agents
cd /tmp/agency-agents && bash scripts/install.sh --tool claude-code

# 2. Everything-Claude-Code (61 Agent + 246 Skills + workflows)
git clone https://github.com/affaan-m/everything-claude-code.git ~/.claude/everything-claude-code

# 3. Ruflo (多Agent编排)
curl -fsSL https://cdn.jsdelivr.net/gh/ruvnet/ruflo@main/scripts/install.sh | bash

# 4. humanizer-zh (中文AI去痕)
git clone https://github.com/op7418/Humanizer-zh.git ~/.claude/skills/humanizer-zh

# 5. humanizer-en (英文AI去痕)
git clone https://github.com/blader/humanizer.git ~/.claude/skills/humanizer-en

# 6. claude-cwc (可视化Agent工作流编排)
npm install -g claude-cwc

# 7. n8n-agent-loop (Claude Code Agent在n8n中)
npm install -g n8n-nodes-claude-code-agent-loop

# 8. CodeGraph (代码知识图谱MCP)
npm install -g @colbymchenry/codegraph
git clone https://github.com/colbymchenry/codegraph.git ~/.claude/codegraph

# 9. TaskMarket CLI (Agent赏金平台)
npm install -g @lucid-agents/taskmarket
taskmarket init  # 注册Agent身份

# 10. npm包（在项目目录内）
cd auto-token-earn
npm install @ddmnpm/humanizer-zh

# 11. Hermes Agent (761 skills + 自学习引擎)
git clone https://github.com/NousResearch/hermes-agent.git ~/.claude/hermes-agent

# 12. mem0 (向量记忆层)
npm install -g mem0ai --ignore-scripts

# 13. Browser-Use (网页自动化) — 需要先装 Python 3.10+
# 安装 Python 后执行：
pip install browser-use
playwright install  # 安装浏览器驱动
```
