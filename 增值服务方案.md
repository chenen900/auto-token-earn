## 怎么加入工具箱

在工具箱页面加一个简单的会员入口：

```
首页三张卡旁边 → 第四张卡："解锁专业版"
    ↓
点击 → 简单注册 → 选择套餐 → 微信/支付宝支付
    ↓
解锁：批量审查 | 竞品分析 | 市场数据 | API Key
```

不需要复杂用户系统——邮箱注册即可，支付用微信/支付宝扫码。

## 付费点位设计

```
免费用户用工具箱
    │
    ├── 合规审查：好用！但每次只能查一条
    │   └── → 会员：一次查 100 条，导出报告
    │
    ├── 利润计算：好用！但不知道竞品卖多少
    │   └── → 专业版：看市场数据，智能定价
    │
    └── 合作伙伴链接：走我们的链接注册 → 我们赚佣金
```

**三条收入线：会员费 + 专业版费 + 合作伙伴佣金。**
