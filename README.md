# Auto Token Earn 项目

> 目标：用 AI Agent 自动化赚取收益，覆盖每日 Token 消耗，最终实现正向现金流。
> 核心原则：全自动、不接触人、不需要编程能力兜底。

---

## 每日 Token 消耗估算

| 使用强度 | 输入 Token | 输出 Token | 估算费用 |
|----------|-----------|-----------|---------|
| 轻度（1-2 个简单任务） | ~100K | ~30K | $1-3/天 |
| 中度（3-5 个 Agent 协作） | ~500K | ~150K | $5-10/天 |
| 重度（多工作流并行） | ~1M-2M | ~300K-500K | $15-30/天 |

**设定目标：日均赚取 $10-20，覆盖中度使用 + 留出利润。**
